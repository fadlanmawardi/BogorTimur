Apa perbedaan antara margin dan padding pada CSS? menjelaskan dengan contoh Margin adalah ruang di luar batas elemen. Ini digunakan untuk menciptakan jarak antara elemen dengan elemen lainnya. Margin tidak memiliki latar belakang dan transparan. contoh dibawah ini .box { margin: 20px; /* memberikan jarak 20px di semua sisi */ background-color: lightblue } Padding adalah ruang dalam elemen, antara konten elemen dan batasnya. Padding memiliki latar belakang yang sama dengan elemen itu sendiri. contoh dibawah ini .box { padding: 20px; /* Memberikan jarak 20px di dalam batas elemen */ background-color: lightcoral; }
